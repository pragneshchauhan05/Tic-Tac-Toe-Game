# 🎮 Tic Tac Toe Game

A simple and fun **Tic Tac Toe (X-O)** game built using **HTML, CSS, and JavaScript**.  
This project is perfect for beginners to learn basic web development and JavaScript game logic.

---

## 🚀 Features

- 👥 Two-player game (X vs O)
- 🧠 Win detection (row, column, diagonal)
- 🤝 Draw detection
- 🔄 Restart game option
- 📱 Responsive design (mobile + desktop)
- 🎨 Clean and simple UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎮 How to Play

1. Open the project in browser  
2. Player X starts first  
3. Click any empty box to mark X or O  
4. Players take turns  
5. First to match 3 in a row wins  
6. Click restart to play again  

---

## 💡 What You Learn

- DOM Manipulation  
- Event Listeners  
- Game Logic Building  
- CSS Styling  
- Basic JavaScript Programming
